# 🚀 AI Web Scraper

An intelligent web scraping tool that combines Selenium-based browser automation with local LLMs (Ollama) to extract structured information from websites using natural language prompts.

---

## ✨ Features

- 🌐 Scrapes dynamic websites using Selenium  
- 🧠 AI-powered data extraction with Ollama (Llama)  
- 🔍 Natural language parsing (describe what to extract)  
- 🧩 DOM chunking for large pages  
- 🧹 HTML cleaning for improved accuracy  
- ⚡ Interactive UI built with Streamlit  
- 🛡 Bright Data integration for reliable scraping  

---

## 🏗 How It Works

1. Enter a website URL  
2. The app scrapes and cleans the webpage content  
3. Describe the data you want to extract  
4. Ollama processes the content and returns relevant information  

---

## 🧰 Tech Stack

- **Frontend:** Streamlit  
- **LLM:** LangChain + Ollama (Llama 3.2)  
- **Scraping:** Selenium, BeautifulSoup4  
- **Parsing:** lxml, html5lib  
- **Environment:** python-dotenv  
- **Proxy:** Bright Data
